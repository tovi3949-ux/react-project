# Helpdesk Support System

A comprehensive ticket management system built with React, TypeScript, and Material UI. This system allows customers to open support tickets, agents to manage them, and admins to oversee the entire operation.

### User Roles
* **Customer:**
    * Sign up and Login.
    * Open new support tickets.
    * View tracking history of their tickets.
    * Chat with support agents on specific tickets.
* **Agent:**
    * View and manage tickets assigned to them.
    * Update ticket status and priority.
    * Respond to customer tickets via chat.
* **Admin:**
    * Full dashboard overview.
    * Manage users (Add, Edit, Delete).
    * Manage all tickets in the system.
    * Configure system settings (Statuses, Priorities).
